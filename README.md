## Installation

* MONGO_DB & mongoose
```sh
$ npm install --save  mongoose
```
* express
```sh
$ npm install --save  express
```
* ejs
```sh
$ npm install --save  ejs
```
* bodyParser
```sh
$ npm install --save  body-parser
```
* method-override
```sh
$ npm install --save method-override
```
